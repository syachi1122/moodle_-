Concept

学習体験に、洗練という＋αを。

moodle αは、学生が毎日向き合うMoodleを、より自然で、より快適で、より現代的な体験へ導くための拡張です。

01

迷わないUI

情報の整理と余白を見直し、必要な機能へすばやくたどり着けるダッシュボードを目指します。

02

見逃さない締切

近い課題を強く、重要な予定を自然に。締切の把握を、感覚的に分かる表示へ変えます。

03

日常に溶け込む導線

スマホ通知やデスクトップショートカットを通して、Moodleをもっと身近な存在にします。

Features

学生の毎日に、必要な機能だけを。

ただ機能を増やすのではなく、学習の負担を減らすための価値に絞って設計しています。

GC

Googleカレンダー連携

課題や予定をGoogleカレンダーに取り込み、スマートフォンで通知。提出忘れの防止をサポートします。

スマホで課題を受け取る
UI

近代的で分かりやすいUI

情報の優先度を整理し、見たいものを迷わず確認できるダッシュボードへ。毎日の操作がより快適になります。

必要な情報にすぐ届く
DL

締め切り表示を強化

締切の近い課題を見やすく整理し、危険な予定をひと目で把握。今日やるべきことがすぐ分かります。

締切を直感的に把握
PC

デスクトップにショートカット追加

PCからMoodleへ最短でアクセス。毎回検索する手間を減らし、日常の導線をすっきりまとめます。

ワンクリックでMoodleへ
Showcase

導入後の景色が、学習の印象を変える。

見やすく整理された締切表示とカレンダーの統合で、課題管理をもっと自然なものにします。

締切一覧

近い課題から、優先順位が分かる。

赤・黄・青の視認性を活かした表示で、今すぐ取り組むべき課題が自然と目に入ります。

カレンダー連携

予定と課題を、ひとつの流れで。

Googleカレンダーとの連携により、Moodleの情報を日常のスケジュール管理へなめらかにつなげます。

導線の改善

欲しい場所へ、すばやく移動。

マイコースや検索などの入口を整理し、PCでもスマホでもストレスの少ない導線を実現します。

Roadmap

これからのmoodle α。

すでに便利なだけで終わらず、今後も学生の使いやすさを中心に機能を進化させていきます。

授業表示を時間割式に

マイコースの一覧を、より学校生活に近い時間割ベースの見せ方へ。必要な授業へ直感的にアクセスできる構成を検討中です。

ダークモード対応

夜間の学習や長時間利用でも見やすく、目にやさしい表示へ。サイト全体の雰囲気を保ちながらの最適化を予定しています。
